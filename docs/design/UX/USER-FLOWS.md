## 1. 首次使用流程

```
┌──────────────┐
│   打开应用    │
└──────┬───────┘
       │
       ▼
┌──────────────┐
│   角色选择    │
│  ┌──┬──┐    │
│  │👨‍👩‍👧│👶│    │
│  └──┴──┘    │
└──────┬───────┘
       │
       ├───────────────┐
       │               │
       ▼               ▼
┌──────────────┐ ┌──────────────┐
│   家长端     │ │   孩子端     │
│   登录/注册   │ │   扫码绑定   │
└──────┬───────┘ └──────┬───────┘
       │               │
       ▼               ▼
┌──────────────┐ ┌──────────────┐
│  家长首页     │ │  今日任务页   │
│  (查看孩子)   │ │  (横屏+番茄钟)│
└──────────────┘ └──────────────┘
```

**流程说明：**
1. 用户首次打开应用，进入角色选择页
2. 选择「家长」→ 手机号登录/注册 → 进入家长首页
3. 选择「孩子」→ 扫码绑定家长账号 → 进入今日任务页

```

   │                        ├→ 满 10 ⭐ → 自动兑换 1 🌙
   │                        │   显示: "🎉 10颗星星兑换成1个月亮!"
```

## 🔀 异常流程

### 任务逾期处理

```
任务逾期 (当前时间 > 结束时间 + 15分钟)
       │
       ▼
┌─────────────────────────────────────────┐
│  家长端显示:                            │
│  • 任务标记为 "已逾期"                  │
│  • 推送通知给家长                        │
│  • 孩子端不显示逾期任务                  │
│                                         │
│  可选操作:                              │
│  [延后到今天] [标记完成] [删除]         │
└─────────────────────────────────────────┘
```

### 网络中断处理

```
网络中断
   │
   ▼
┌─────────────────────────────────────────┐
│  • 本地保存操作记录                      │
│  • 显示 "离线模式" 提示                  │
│  • 允许继续本地操作                      │
│  • 网络恢复后自动同步                    │
└─────────────────────────────────────────┘
```
